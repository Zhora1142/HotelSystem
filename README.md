# HotelSystem

**HotelSystem** — это курсовой проект по дисциплине **«Конструирование Програмного Обеспечения»**.

## Тема проекта

Разработка информационной системы для управления гостиницей.  
Проект предназначен для автоматизации основных процессов гостиницы: работы с клиентами, номерами, бронированиями и дополнительными услугами.

## Возможности системы

- авторизация пользователя;
- ведение базы клиентов;
- добавление и управление номерами;
- управление типами номеров;
- создание и сопровождение бронирований;
- оформление заселения и контроль статусов брони;
- добавление дополнительных услуг к бронированию;
- формирование отчетов по загрузке и операциям.

## Стек проекта

- `C#`
- `.NET`
- `WPF`
- `Entity Framework Core`
- `PostgreSQL`

## Назначение

Проект создан в учебных целях как курсовая работа и демонстрирует применение многослойной архитектуры:

- `HotelSystem.Domain` — доменная модель;
- `HotelSystem.DAL` — доступ к данным;
- `HotelSystem.BLL` — бизнес-логика;
- `HotelSystem.WPF` — пользовательский интерфейс.
